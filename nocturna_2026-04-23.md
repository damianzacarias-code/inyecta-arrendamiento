```

├── index.ts            # factory getExtractProvider() + cache + reset
├── types.ts            # IExtractProvider, TipoExtract, TIPOS_EXTRACT
├── schemas.ts          # 5 Zod schemas + computeConfidence + getSchemaForTipo

```

5 tipos: `CSF`, `INE`, `COMPROBANTE_DOMICILIO`, `FACTURA_BIEN`,
`ACTA_CONSTITUTIVA`. Todos los campos `.nullable().optional()`.

`ClaudeProvider`:
- Manda PDF como `document`/imagen como `image` content block.
- Parser tolerante: JSON crudo, fenced code blocks,
  extracción balanceada por llaves.
- Reintenta 1× con prompt más estricto si el JSON falla.
- Si la validación Zod falla, baja confidence en 0.5 (no rechaza
  los datos — el frontend decide si los usa).
